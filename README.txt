What you need to compile the project :
- bullet 2.81 with Extras (need for HACD)
- qt5 with qt5xmlPatterns

Package name to install on Ubuntu 13.10 based distribution:
- libbullet-dev
- libbullet-extras-dev
- qt5-default
- libqt5xmlpatterns5-dev
- freeglut3-dev

To compile the project:
- move to src/
- execute qmake
- execute make

To run the project:
- move to build/
- execute ./gui
